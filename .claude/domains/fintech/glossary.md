# 금융 용어집

## 결제 관련

| 용어 | 영문 | 설명 |
|------|------|------|
| 승인 | Authorization | 카드 결제 가능 여부 확인 및 금액 확보 |
| 매입 | Capture | 승인된 금액을 실제로 청구하는 과정 |
| 취소 | Cancel/Void | 매입 전 승인 취소 (당일 취소) |
| 환불 | Refund | 매입 후 금액 반환 |
| 부분취소 | Partial Refund | 일부 금액만 취소/환불 |
| 망상취소 | Same-day Cancel | 승인 당일 카드사 전산망 내 취소 |

## 정산 관련

| 용어 | 영문 | 설명 |
|------|------|------|
| 정산 | Settlement | 가맹점에 결제 대금 지급 |
| D+N | D+N Settlement | 거래일(D) 기준 N일 후 정산 |
| 수수료 | Commission/Fee | 결제 처리 비용 |
| 대사 | Reconciliation | 내부 데이터와 외부 데이터 매칭 확인 |
| 차감 | Deduction | 정산금에서 수수료 등 공제 |

## 카드 관련

| 용어 | 영문 | 설명 |
|------|------|------|
| PAN | Primary Account Number | 카드 번호 |
| CVV/CVC | Card Verification Value | 카드 뒷면 보안 코드 |
| BIN | Bank Identification Number | 카드번호 앞 6자리 (발급사 식별) |
| 유효기간 | Expiry Date | 카드 만료일 (MM/YY) |
| 할부 | Installment | 결제금액 분할 납부 |

## 업계 용어

| 용어 | 영문 | 설명 |
|------|------|------|
| PG | Payment Gateway | 온라인 결제 중개 서비스 |
| VAN | Value Added Network | 카드사-가맹점 간 네트워크 |
| 가맹점 | Merchant | 결제를 받는 판매자 |
| 카드사 | Card Issuer | 카드 발급 금융사 |
| 매입사 | Acquirer | 가맹점 결제 대행사 |

## 인증 관련

| 용어 | 영문 | 설명 |
|------|------|------|
| JWT | JSON Web Token | 인증 정보를 담은 토큰 |
| Access Token | Access Token | API 접근용 단기 토큰 |
| Refresh Token | Refresh Token | Access Token 갱신용 장기 토큰 |
| Token Rotation | Token Rotation | 보안을 위한 토큰 갱신 기법 |
| MFA/2FA | Multi-Factor Auth | 다중 인증 |

## 보안/규정

| 용어 | 영문 | 설명 |
|------|------|------|
| PCI-DSS | Payment Card Industry Data Security Standard | 카드 데이터 보안 표준 |
| 전금법 | Electronic Financial Transactions Act | 전자금융거래법 |
| KYC | Know Your Customer | 고객 신원 확인 |
| AML | Anti-Money Laundering | 자금세탁 방지 |
| FDS | Fraud Detection System | 이상거래 탐지 시스템 |

## 기술 용어

| 용어 | 영문 | 설명 |
|------|------|------|
| 멱등성 | Idempotency | 동일 요청 반복 시 동일 결과 보장 |
| 웹훅 | Webhook | 이벤트 발생 시 자동 알림 호출 |
| Circuit Breaker | Circuit Breaker | 연속 실패 시 요청 차단 패턴 |
| Rate Limiting | Rate Limiting | 요청 수 제한 |
| 대기열 | Queue | 비동기 처리를 위한 메시지 큐 |

## 상태 용어

| 용어 | 영문 | 설명 |
|------|------|------|
| 대기 | Pending | 처리 대기 상태 |
| 처리중 | Processing | 처리 진행 중 |
| 완료 | Completed | 처리 완료 |
| 실패 | Failed | 처리 실패 |
| 만료 | Expired | 유효 기간 초과 |

## 약어

| 약어 | 전체 | 설명 |
|------|------|------|
| TID | Transaction ID | 거래 고유 식별자 |
| MID | Merchant ID | 가맹점 고유 식별자 |
| OTP | One-Time Password | 일회용 비밀번호 |
| API | Application Programming Interface | 응용 프로그램 인터페이스 |
| TPS | Transactions Per Second | 초당 처리 건수 |
