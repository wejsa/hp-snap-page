# 상품 관리

## 개요

이커머스 서비스의 상품 등록, 카탈로그, 카테고리, 옵션 관리를 설명합니다.

## 상품 구조

### 상품 모델

```
Product (상품)
  ├── Category (카테고리, 다단계)
  ├── Option (옵션: 색상, 사이즈 등)
  │     └── OptionValue (옵션값: 빨강, L 등)
  ├── Variant (옵션 조합별 SKU)
  │     ├── price (판매가)
  │     ├── stock (재고)
  │     └── sku (고유 코드)
  └── Image (상품 이미지, 대표/상세)
```

### 상품 상태

```
DRAFT → ACTIVE → SOLD_OUT → DISCONTINUED
  │                  ↑
  └── 임시저장     재고 0 시 자동 전환
```

| 상태 | 설명 | 노출 |
|------|------|------|
| DRAFT | 임시 저장 | 비노출 |
| ACTIVE | 판매 중 | 노출 |
| SOLD_OUT | 품절 | 노출 (품절 표시) |
| DISCONTINUED | 판매 중단 | 비노출 |

## 카테고리

### 다단계 카테고리 구조

```
의류 (depth 1)
  ├── 상의 (depth 2)
  │     ├── 티셔츠 (depth 3)
  │     └── 셔츠 (depth 3)
  └── 하의 (depth 2)
        ├── 바지 (depth 3)
        └── 스커트 (depth 3)
```

- 최대 깊이: 3단계
- Materialized Path 또는 Closure Table 패턴 사용

## 가격 정책

| 항목 | 규칙 |
|------|------|
| 금액 단위 | BigDecimal (정밀도 보장) |
| 통화 | KRW (원화, 소수점 없음) |
| 할인가 | 정가 × (1 - 할인율), 1원 단위 절사 |
| 최소 판매가 | 100원 이상 |

## 검색

### 검색 대상 필드
- 상품명, 브랜드, 카테고리명, 태그

### 정렬 옵션
- 인기순 (판매량 기준)
- 최신순 (등록일 기준)
- 가격순 (낮은/높은)
- 평점순
