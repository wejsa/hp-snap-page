# 회원 관리

## 개요

이커머스 서비스의 회원 가입, 로그인, 마이페이지 기능을 설명합니다.

## 회원 가입

### 필수 정보
- 이메일 (고유, 로그인 ID)
- 비밀번호 (bcrypt 해싱, cost 12 이상)
- 이름
- 연락처 (본인 인증 후)

### 가입 플로우

```
1. 이메일 중복 확인
2. 비밀번호 유효성 검증 (8자 이상, 대소문자+숫자+특수문자)
3. 약관 동의 확인 (필수/선택 구분)
4. 본인 인증 (휴대폰/이메일)
5. 계정 생성 + 환영 쿠폰 발급
```

## 로그인

### 인증 방식
- 이메일 + 비밀번호 기반
- 소셜 로그인 (OAuth 2.0): 카카오, 네이버, 구글
- JWT 기반 세션 관리 (Access Token + Refresh Token)

### 보안 정책
- 로그인 실패 5회 → 계정 잠금 (30분)
- 비밀번호 변경 시 기존 세션 전체 무효화
- 휴면 계정 전환: 1년 미접속 시

## 마이페이지

### 기능 목록

| 기능 | 설명 |
|------|------|
| 주문 내역 | 주문 목록/상세/재주문 |
| 배송 추적 | 실시간 배송 상태 |
| 취소/반품 | 주문 취소, 반품 신청 |
| 쿠폰함 | 보유 쿠폰 조회/사용 |
| 포인트 | 적립/사용 내역 |
| 찜 목록 | 관심 상품 관리 |
| 개인정보 | 회원 정보 수정 |
| 배송지 관리 | 배송지 CRUD |

### 개인정보 처리

| 항목 | 처리 방식 |
|------|----------|
| 비밀번호 | bcrypt 단방향 해싱 |
| 연락처 | AES-256 양방향 암호화 |
| 주소 | AES-256 양방향 암호화 |
| 카드 정보 | 저장 금지 (PG사 토큰만 저장) |

## 회원 탈퇴

```
1. 본인 인증 확인
2. 탈퇴 사유 수집
3. 미완료 주문 확인 (있으면 탈퇴 불가)
4. 개인정보 즉시 삭제 (또는 법적 보관 기간 후)
5. 포인트/쿠폰 소멸 안내
```
